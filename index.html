Projects

Cards Against Humanity

A digital version of a popular card game, Cards Against Humanity, used at social events with close friends. The game supports 4 players max. Player one starts out as a judge, and turns for judge rotate to each player each round. Score is kept for all players. A notification is brought up after round 5 to end the game. If the group wishes to continue, the game will continue until the black card deck is depleted.

Pwn Hub

PwnHub is a website for users to share their latest video game triumphs. It best thrives on Xbox One because of Microsoft's built in DVR function. The input will take in a video link and a description of the video. Other users will be able to comment on posted videos. These functions are only accessible when a user has an account and logged in.

Groupy

Groupy is an app that allows people with similar taste in music group together and attend events. It allows a platform for users to add events to their individual user profile. Within the user profile, each event displays information about the event along with the ability to comment and in turn band together a group to attend the venue.

Cover Maestro

Cover Maestro is a website for users with little to no music experience to pick up an instrument (guitar for now), and start playing their favorite pop songs. On signup, users are taken to welcome page to learn their first lesson. Once a user has joined, they have the ability to add their own rendition of any pop song or edit existing posted songs.
